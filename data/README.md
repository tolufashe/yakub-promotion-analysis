# Data Directory

This directory contains all data files for the Yakub Promotion Analysis project.

## Structure

```
data/
├── raw/           # Original, immutable data
├── processed/     # Cleaned and transformed data
└── README.md      # This file
```

## Raw Data (`raw/`)

**DO NOT MODIFY** files in this directory. These are the original data files.

### Files

| File | Description | Size | Records |
|------|-------------|------|---------|
| `Promotion Dataset.csv` | Employee promotion records | ~5 MB | 38,312 |

### Data Dictionary

| Column | Description | Type | Values |
|--------|-------------|------|--------|
| `EmployeeNo` | Unique employee identifier | String | YAKUB-XXXX |
| `Division` | Department/division | Categorical | 6 divisions |
| `Qualification` | Highest education level | Categorical | Non-University, First Degree, MSc/MBA/PhD |
| `Gender` | Employee gender | Binary | Male, Female |
| `Channel_of_Recruitment` | Recruitment source | Categorical | 4 channels |
| `Trainings_Attended` | Number of trainings | Numeric | Integer |
| `Year_of_birth` | Birth year | Numeric | 1948-2000 |
| `Last_performance_score` | Previous performance | Numeric | 0-14 |
| `Year_of_recruitment` | Join year | Numeric | 1984-2018 |
| `Targets_met` | Target achievement | Binary | 0, 1 |
| `Previous_Award` | Prior awards | Binary | 0, 1 |
| `Training_score_average` | Training performance | Numeric | 0-100 |
| `State_Of_Origin` | State of origin | Categorical | 37 states |
| `Foreign_schooled` | Abroad education | Binary | Yes, No |
| `Marital_Status` | Marital status | Categorical | Single, Married, Divorced |
| `Past_Disciplinary_Action` | Disciplinary history | Binary | Yes, No |
| `Previous_IntraDepartmental_Movement` | Dept transfers | Binary | Yes, No |
| `No_of_previous_employers` | Previous companies | Numeric | Integer |
| `Promoted_or_Not` | **Target variable** | Binary | 0, 1 |

### Data Quality

- **Missing Values**: 1,679 missing in `Qualification` (4.4%)
- **Duplicates**: None detected
- **Target Distribution**: 8.5% promoted, 91.5% not promoted

## Processed Data (`processed/`)

Files in this directory are generated by the data processing pipeline and can be regenerated.

### Files

| File | Description | Generated By |
|------|-------------|--------------|
| `train_features.csv` | Training features (after encoding) | `src/feature_engineering.py` |
| `test_features.csv` | Testing features (after encoding) | `src/feature_engineering.py` |
| `train_target.csv` | Training target | `src/data_processing.py` |
| `test_target.csv` | Testing target | `src/data_processing.py` |

## Data Usage

### Loading Raw Data

```python
import pandas as pd

# Load raw data
df = pd.read_csv('data/raw/Promotion Dataset.csv')

# View first few rows
print(df.head())

# Get summary statistics
print(df.describe())
```

### Data Processing Pipeline

```python
from src.data_processing import load_and_clean_data, handle_missing_values
from src.feature_engineering import create_features, prepare_features

# Load and clean
df = load_and_clean_data('data/raw/Promotion Dataset.csv')
df = handle_missing_values(df)

# Create features
df = create_features(df)

# Full pipeline (includes split, encoding, scaling)
X_train, X_test, y_train, y_test, scaler = prepare_features(df)
```
